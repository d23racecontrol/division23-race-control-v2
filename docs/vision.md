# PHOENIX
## Division 23 United Headquarters

## Vision

Phoenix ist die zentrale Plattform für Division 23 United.

Das Projekt vereint alle organisatorischen Bereiche an einem Ort und ersetzt langfristig einzelne Werkzeuge und Webseiten.

Phoenix besteht aus folgenden Modulen:

- 🏁 Race Control
- 🎉 Events
- 🎓 Rookie Academy
- 🏎 Division 23 eSport
- 👥 Community
- 🤝 Sponsoren
- 📊 Analytics
- 📁 Media Center
- ⚙ Administration

## Grundprinzipien

1. Eine Plattform statt vieler Einzellösungen.
2. Eine Person wird nur einmal angelegt.
3. Öffentliche und interne Bereiche sind klar getrennt.
4. Daten werden zentral in Supabase gespeichert.
5. Race Control bleibt das Herzstück der Plattform.
6. Erweiterungen werden modular entwickelt.
7. Jede Änderung wird dokumentiert.

## Ziel

Eine moderne, übersichtliche und leistungsfähige Plattform, die die komplette Organisation von Division 23 United unterstützt.
