# Phoenix – Division 23 Headquarters

Phoenix wird das zentrale Hauptquartier von Division 23 United.

## Ziel

Phoenix soll die bestehende Race Control schrittweise erweitern und später alle wichtigen Bereiche an einem Ort zusammenführen:

- Race Control für Ligen, Fahrer, Ergebnisse und Tabellen
- Login und verschiedene Benutzerrollen
- Cloud-Speicherung über Supabase
- Teambereiche für Division 23 eSport
- Bewerbungen und Fahrerprofile
- Kalender und weitere Community-Module
- Öffentlicher Zuschauerbereich

## Wichtig

Die bestehende Race Control bleibt während der Entwicklung vollständig funktionsfähig. Phoenix wird Schritt für Schritt daneben aufgebaut und anschließend sicher mit der Race Control verbunden.
